Smart Chef: Генератор рецептів по фото інгредієнтів

Flutter Gemini Vision API Image Recognition
[Скріншот: Аналіз вмісту холодильника]

Проблема

Користувачі часто стикаються з проблемою "що приготувати сьогодні?", маючи обмежений набір продуктів. Харчові відходи зростають через те, що люди не знають, як використати залишки інгредієнтів.

Рішення

Додаток використовує Gemini Vision API для розпізнавання об'єктів на фото. Користувач просто фотографує продукти на столі, а ШІ миттєво ідентифікує їх та пропонує список рецептів, які можна приготувати "тут і зараз".

Технічні виклики

  • Точність розпізнавання: Налаштування промптів для диференціації схожих продуктів (наприклад, різних видів сиру або соусів).
  • Креативність рецептів: Використання системних інструкцій для того, щоб ШІ пропонував не лише стандартні страви, а й креативні варіанти на основі поєднання смаків.
