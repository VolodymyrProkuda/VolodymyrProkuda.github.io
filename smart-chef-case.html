Smart Chef: Генератор рецептів по фото інгредієнтів

Flutter Gemini Vision API Image Recognition Clean Architecture

Проблема

Користувачі часто стикаються з питанням "що приготувати сьогодні?", маючи обмежений набір продуктів. Харчові відходи зростають через те, що люди не знають, як ефективно використати залишки інгредієнтів у холодильнику.

Рішення

Додаток використовує потужність Gemini Vision API для розпізнавання об'єктів на фото. Користувач просто фотографує продукти, а ШІ миттєво ідентифікує їх, враховує їхній стан та пропонує персоналізований список рецептів, які можна приготувати негайно.

Технічні виклики та реалізація

  • Точність розпізнавання: Впроваджено систему уточнюючих промптів для диференціації схожих продуктів (наприклад, різних сортів томатів або типів борошна).
  • Креативність рецептів: Використання системних інструкцій (System Instructions) дозволило ШІ пропонувати не лише стандартні страви, а й незвичайні поєднання на основі молекулярної сумісності смаків.
  • Швидкість UI: Оптимізація обробки зображень перед відправкою на сервер для зменшення затримки та економії трафіку користувача.

Чому цей проєкт важливий?

Цей кейс демонструє навички роботи з Multi-modal AI. Це не просто чат-бот, а інструмент, який взаємодіє з реальним фізичним світом через камеру смартфона, вирішуючи щоденну побутову проблему за допомогою сучасних технологій.
